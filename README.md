# Shorts.WTF Frontend
This repository contains the [Shorts.wtf](https://shorts.wtf/).

## Tech Stack
- ReactJS
- NextJS
- Ant Design
- ethers
- Web3Modal
- Wagmi
- Styled components
- React testing library
- Jest


## Installation and Setup Instructions

Clone down this repository. You will need `node` version `">=18"` and `yarn` version `">=1"` installed globally on your machine.

### Installation:

`yarn`

### To Run Test Suite:

`yarn test`

### To Start Server:

- In one terminal:

`node test_server.js`

- In another terminal:

`yarn dev`

- Then visit the app:

`localhost:3000`

### To build:

`yarn run build`
